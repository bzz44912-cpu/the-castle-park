# The Loft Lounge Website

Ce dossier contient le code source du site web pour **The Loft Lounge** à Sidi Bouzid.

## Structure du projet

- `index.html` : La page principale du site.
- `styles.css` : Les styles CSS pour le design sombre et premium.
- `script.js` : Le code JavaScript pour les interactions (menu mobile, scroll).
- `assets/` : Dossier contenant les images (image de fond générée).

## Comment utiliser

1. Ouvrez simplement le fichier `index.html` dans votre navigateur web préféré (Chrome, Firefox, Edge, etc.).
2. Vous pouvez modifier les textes et les images directement dans le code HTML pour mettre à jour le contenu.

## Personnalisation

- Pour changer les couleurs, modifiez les variables CSS dans `styles.css` (section `:root`).
- Pour ajouter vos propres images, placez-les dans le dossier `assets/images` et mettez à jour les liens dans `index.html`.
